Git and Github





Git and GitHub are related tools used in software development, but they serve different purposes:

Git

  • What it is: A distributed version control system.
  • Purpose: Tracks changes to files (usually code) over time.

Key features

  • Works locally on your computer
  • Lets you commit, branch, merge, revert, etc.
  • Helps manage code history, collaboration, and backups.

Git-Hub

  • What it is: A web-based platform for hosting Git repositories.
  • Purpose: Makes it easier to collaborate using Git.

Key features

  • Cloud storage for Git repos.
  • Tools for collaboration (issues, pull requests, reviews).
  • Tools for collaboration (issues, pull requests, reviews).
  • Integrates with CI/CD tools, GitHub Actions, etc.

How they work together

  1. You use Git on your local machine to track changes.
  2. You use GitHub to share that code and collaborate with others.
